# 🎨 Tarjeta de Perfil Interactiva

Tarjeta de presentación con cambio de tema claro/oscuro, construida con HTML, CSS y JavaScript puro.

🔗 **[Ver demo en vivo](https://tunombre.github.io/tarjeta-perfil)**

## Capturas

![Modo claro](screenshots/Modo%20claro.png)
![Modo oscuro](screenshots/Modo%20oscuro.png)

## ¿Qué hace?

- Muestra una tarjeta de perfil con nombre, descripción y habilidades
- Botón que alterna entre modo claro y modo oscuro
- Transiciones suaves al cambiar de tema

## Tecnologías

- HTML5 semántico
- CSS3 (variables CSS / custom properties)
- JavaScript vanilla (manipulación del DOM)

## Qué aprendí

Este fue mi primer proyecto fuera del curso. Practiqué cómo conectar HTML, CSS y JS entre sí, cómo manipular el DOM con `classList`, y cómo usar variables CSS para manejar temas sin duplicar código.

## Cómo correrlo localmente

```bash
git clone https://github.com/tunombre/tarjeta-perfil.git
cd tarjeta-perfil
```

Abrí `index.html` en tu navegador. No necesita instalación ni dependencias.
